A modified and improved tramway system for FiveM, based on glitchdetector's [original tramway resource](https://github.com/glitchdetector/fivem-aerial-tramway). Uses ox_lib zones to reduce the performance impact of the original resource and improve tramway cable car syncing.

# Features
- Cable cars move up and down the Pala Springs Aerial Tramway lines 
- Syncing between players during landing at the lower platform
- Cars can be ridden by entering them and pressing E while they are at a station
- Audio for tram movements, doors opening/closing, and the arrival/departure cues
- Uses ox_lib zones to reduce performance impact on players not near the tramway

# Dependencies
- [ox_lib](https://github.com/overextended/ox_lib)
- Optional: [ox_target](https://github.com/overextended/ox_target) if using ox_target instead of text UI interactions
